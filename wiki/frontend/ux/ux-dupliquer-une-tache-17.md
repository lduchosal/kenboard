# #17 — UX / Dupliquer une tâche

L'inzterface utilisateur permet de duplique une tâche avec un bouton "Dupliquer" dans le formulaire d'édition d'une tâche.

La tâche existante est copiée dans un nouvelle tâche :
- titre : titre - copy
- contenu
- responsable
- délai 
- projet
- status (selon la colonne posée)

Le formulaire d'édition reste ouvert, l'utilisateur peut modifier la nouvelle tâche selon le formulaire standard.
