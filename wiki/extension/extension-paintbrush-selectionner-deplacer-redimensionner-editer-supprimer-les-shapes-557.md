# #557 — EXTENSION / paintbrush — sélectionner / déplacer / redimensionner / éditer / supprimer les shapes

UX d'édition des shapes existants en mode paintbrush :
- Sélection : clic sur un rectangle ou un texte = sélectionne le shape.
- Drag du body : déplace le shape.
- Drag d'un coin : redimensionne le rectangle.
- Clic sur un texte : ouvre le composer pour éditer son contenu inline.
- Touche Del/Backspace : supprime le shape sélectionné.

---

## Annulée

Décision produit : le flux actuel (supprimer via le drawer + redessiner) est jugé suffisant. Pas la peine d'ajouter un outil de sélection + handles de resize + edit inline pour cette itération. La simplicité du paintbrush (3 outils ultra-limités) prime sur la richesse d'édition.

À ré-ouvrir si un cas d'usage concret demande de l'édition fine plus tard.
